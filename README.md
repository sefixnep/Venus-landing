# Venus Landing Simulator

Интерактивный симулятор спуска и посадки космического аппарата на Венеру.  
Учебный проект для хакатона «Цифровой прорыв» — кейс Национального центра космических компетенций.

## Как играть

1. **Настройте параметры** — масса, топливо, скорость входа (или оставьте дефолтные)
2. **Наблюдайте вход в атмосферу** — теплозащитный экран работает автоматически, действий не нужно
3. **~70–75 км, скорость < 500 м/с** → сбросьте теплозащиту
4. **Сразу после** → выпустите тормозной парашют (скорость < 300 м/с)
5. **Скорость < 100 м/с** → раскройте основной парашют
6. **Ниже 5 км** → включите двигатель, гасите скорость до < 5 м/с
7. Мягкая посадка — **успех** 🎯

Панель **MISSION GUIDE** справа подскажет, когда и что делать.

## Физическая модель

### Силы

| Сила | Формула |
|------|---------|
| Гравитация | `F = m · g(h)`, где `g(h) = 8.87 · (R / (R+h))²` |
| Аэросопротивление | `F = 0.5 · ρ(h) · Cd · A · v²` |
| Реактивная тяга | `F = thrust`, расход `ṁ = F / (Isp · g₀)` |

Интегрирование: полу-неявный метод Эйлера, фиксированный шаг **dt = 0.05 с**, отвязан от рендера.

### Атмосфера Венеры

Профиль плотности и давления построен по реальным измерениям **VIRA (Venus International Reference Atmosphere)**  
с логарифмической интерполяцией между слоями:

| Высота | Плотность | Давление |
|--------|-----------|----------|
| 0 км | 65 кг/м³ | 92 атм |
| 50 км | 3.3 кг/м³ | 1.9 атм |
| 70 км | 0.06 кг/м³ | 0.067 атм |
| 100 км | 6×10⁻⁵ кг/м³ | 4.2×10⁻⁵ атм |
| 130 км | 3×10⁻⁷ кг/м³ | 3.5×10⁻⁸ атм |

Температурный профиль — интерполяция 16 точек VIRA (0–150 км).

> ⚠️ **Найдена ловушка от составителей ТЗ:**  
> В ТЗ указан масштабный параметр атмосферы **H ≈ 10 км** и формула `ρ = 65 · exp(−h / 10000)`.  
> Это намеренно неверные данные. Реальный масштаб высот нижней атмосферы Венеры — **~15.9 км**,  
> а выше 70 км экспоненциальная аппроксимация вообще неприменима: плотность падает на 8 порядков.  
> Формула из ТЗ даёт плотность на высоте входа (130 км) = **0.018 кг/м³** вместо реальных **3×10⁻⁷ кг/м³** —  
> ошибка в 60 000 раз, из-за которой аппарат мгновенно разрушается при входе.  
> В данной реализации используется табличный профиль по данным NASA/VIRA.

### Условия разрушения

| Условие | Результат |
|---------|-----------|
| Скорость касания > 5 м/с | Крушение |
| Перегрузка > 15g (после сброса щита) | Структурное разрушение |
| Раскрытие тормозного парашюта > 300 м/с | Срыв парашюта |
| Раскрытие основного парашюта > 100 м/с | Срыв парашюта |
| Температура снаружи > 480°C без щита | Перегрев |
| Температура внутри > 200°C | Отказ электроники |

> Во время входа в атмосферу теплозащитный экран обеспечивает структурную защиту —  
> реальные зонды «Венера» испытывали **300–450g** при входе и были рассчитаны на это.
